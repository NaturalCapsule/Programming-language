What is programming language?

A programming language is a set of rules and instructions used to communicate with computers and tell them what to do. It acts as a bridge between human logic and the computer's machine code (binary (0)s and (1)s), enabling programmers to create software, websites, apps, and other technology. Each language has its own unique syntax, which are the specific grammatical rules that must be followed for the instructions to be understood and executed by the computer.

  • How it works: Programmers write instructions in a high-level language that is easier for-humans to read
  • Purpose: Programming language can't understand english like human.
  • Example: Different languages are used for different purpose; for example Python is beginner-friendly while C++ is for advanced Developers.

How did programming languages evolved

In 1972 Dennis Ritchie made C. A new language that made the programming and computer world evolve

  • Why did Dennis Ritchie made C?

      • - To rewrite the Unix operating system in a higher-level language

  • Why is C important?

      • - C is the Mother of all languages like: (C++, Python, JavaScript, etc...)

      - C is one of the best languages to make Operating System due to low-level compatibility

  • Why the name "C"?

      • - Before C, there was a language called B, created by Ken Thompson at Bell Labs.

      - B was itself derived from an even older language called BCPL, but B was simpler and less powerful.

How Programming Languages Work

  • You write human-readable source code.
  • The code must be translated so the computer can understand it.
  • Translation happens in three main ways.

      • - Interpreters execute code line by line.

      - Compilers convert the entire program into machine code.

      - Hybrid systems use bytecode and Just-In-Time (JIT) compilation.

  • Languages also handle memory (manually, automatically, or through special rules like Rust's ownership).
  • Standard libraries and language rules define how programs behave.
